HTML5 Web Storage API

Learn how to use localStorage and sessionStorage in your web applications

Introduction to Web Storage API

The HTML5 Web Storage API provides mechanisms for browsers to store key/value pairs in a much more intuitive way than using cookies. It offers two main storage mechanisms:

  • localStorage - Stores data with no expiration date
  • sessionStorage - Stores data for one session (data is lost when the browser tab is closed)

Web Storage is more secure and can store larger amounts of data (typically 5-10MB) compared to cookies (4KB). The data is never transferred to the server and is accessible only from the same origin.

localStorage vs sessionStorage

Feature localStorage sessionStorage
Lifetime Persists until explicitly deleted Lasts only for the duration of the browser session
Scope Shared across all tabs/windows with the same origin Limited to the browser tab
Capacity Typically 5-10MB Typically 5-10MB
Accessibility Accessible from any window/tab with same origin Accessible only from the same tab

Code Examples

Basic localStorage Usage

// Setting data localStorage.setItem('username', 'JohnDoe'); localStorage.setItem('theme', 'dark'); // Getting data const username = localStorage.getItem('username'); console.log(username); // Output: JohnDoe // Removing data localStorage.removeItem('theme'); // Clearing all data localStorage.clear();

Basic sessionStorage Usage

// Setting data sessionStorage.setItem('sessionId', 'abc123'); sessionStorage.setItem('pageViews', '5'); // Getting data const sessionId = sessionStorage.getItem('sessionId'); console.log(sessionId); // Output: abc123 // Removing data sessionStorage.removeItem('pageViews'); // Clearing all data sessionStorage.clear();

Storing Objects

// Web Storage only stores strings, so we need to stringify objects const user = { name: 'Jane Smith', email: 'jane@example.com', preferences: { theme: 'light', notifications: true } }; // Store object as string localStorage.setItem('user', JSON.stringify(user)); // Retrieve and parse object const storedUser = JSON.parse(localStorage.getItem('user')); console.log(storedUser.name); // Output: Jane Smith

Browser Compatibility

The Web Storage API is widely supported in all modern browsers, including:

  • Chrome 4+
  • Firefox 3.5+
  • Safari 4+
  • Opera 10.5+
  • Internet Explorer 8+
  • Edge 12+

To check if the browser supports Web Storage:

if (typeof(Storage) !== "undefined") { // Web Storage is supported // Your storage code here } else { // Web Storage is not supported alert("Sorry, your browser doesn't support Web Storage."); }
